# 🎵 Notifications Sounds / 通知音效库

Welcome to the claude-code-notify sound library! Here are some classic 8-bit sounds to make your CLI task completion a bit more rewarding.

Welcome to the claude-code-notify sound library! 欢迎来到 claude-code-notify 音效库！这里有一些经典的 8-bit 音效，让你的 CLI 任务完成更具仪式感。

## 🍄 Available Sounds / 可选音效

| Sound / 音效 | Description / 描述 |
| :--- | :--- |
| [**`minimal_chime.wav`**](minimal_chime.wav) | **Modern & Clean** — Two high-frequency sine waves. Minimalist and non-intrusive. / **极简清脆** — 双频正弦波，现代且不扰人。 |
| [**`mario_coin.wav`**](mario_coin.wav) | **Super Mario Coin** — The iconic NES coin sound. Perfect for quick tasks. / **马里奥金币** — 经典的 8-bit 金币声，适合快速任务。 |
| [**`pokemon_levelup.wav`**](pokemon_levelup.wav) | **Pokémon Level Up** — A melodic arpeggio for when you (or your code) level up. / **宝可梦升级** — 经典的升级旋律，庆祝代码大功告成。 |
| [**`zelda_item.wav`**](zelda_item.wav) | **Zelda Item Get** — A high-pitched fanfar. Ideal for high-importance tasks. / **塞尔达道具获得** — 充满仪式感的 8-bit 齐奏，适合重大进展。 |

## 🛠️ How to use / 如何使用

Open `~/.claude/settings.json` and add the `NOTIFY_DONE_SOUND_FILE` environment variable with the absolute path to your chosen sound.

打开 `~/.claude/settings.json`，在 `env` 字段中添加 `NOTIFY_DONE_SOUND_FILE` 变量，并设置为你所选音效的绝对路径。

## 🎨 Contribute / 贡献音效

Got a cool sound effect? We’d love to have it! Whether it's a retro game chime, a sleek UI ping, or something you synthesized yourself, feel free to submit a Pull Request.

如果你有更好玩的音效，欢迎贡献！无论是复古游戏机音效、现代 UI 提示音，或是你亲手合成的作品，请随时提交 PR。
